# Quick Start

* install python3 and msgpack.

```
sudo apt install python3
pip3 install msgpack
```

* install newest vim.

```
sudo add-apt-repository ppa:jonathonf/vim
sudo apt update
sudo apt install vim
```

* create symlinks.

```
ln -s vimrc ~/.vimrc
ln -s vim ~/.vim
```

* setup plugins using `:PlugInstall` in vim.
